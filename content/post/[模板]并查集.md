## 路径压缩 + 按秩合并

时间复杂度 $ O ( \alpha (n) ) $

```cpp
void init() {
    for (int i = 1; i <= n; i++) f[i] = i, sz[i] = 1;
}

int find(int x) {
    if (x != f[x]) f[x] = find(f[x]);
    return f[x];
}

void merge(int x, int y) {
    x = find(x);
    y = find(y);
    if (x == y) return;
    if (sz[x] > sz[y]) swap(x, y);
    f[x] = y;
    sz[y] += sz[x];
}

bool same(int x, int y) { return find(x) == find(y); }
```
